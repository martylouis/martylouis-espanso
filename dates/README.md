# dates

Text expand dates

## Install

```ssh
espanso install dates --git https://github.com/martylouis/martylouis-espanso --external
```

## Usage

| Trigger | Replace    | Description |
| ------- | ---------- | ----------- |
| `dtt`   | `%Y-%m-%d` | common date |
| `dti`   | `%+`       | ISO Date    |
